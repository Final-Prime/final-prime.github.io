# Security policy

## Reporting

Report a suspected security issue privately to:

finalprime.official@gmail.com

Include the affected public route, a concise reproduction path, expected and
observed behavior, and any material impact. Do not include secrets, personal
data, client information, private source, or destructive proof in a public
issue.

## Scope

The current public website is a static GitHub Pages surface. Contact is a
direct email path, not a live account, authentication, messaging, request
tracking, private workspace, or proposal system.

## Response boundary

No response time, bounty, reward, support term, or disclosure embargo is
promised unless agreed separately in writing. Good-faith reports will be
reviewed when operationally possible.

## Public disclosure

Coordinate public disclosure before publishing exploit details. Do not imply
endorsement, partnership, or official researcher status without written
permission.

The machine-readable contact is published at:

https://final-prime.github.io/.well-known/security.txt
